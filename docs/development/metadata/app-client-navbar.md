# app > clientNavbar

*As of v8.1.*

Navbar definitions.

Example:

```json
{
    "items": {
        "quickCreate": {
            "view": "views/site/navbar/quick-create",
            "class": "dropdown hidden-xs quick-create-container",
            "order": 0,
            "disabled": false
        }
    }
}
```

## items

*Object*

Navbar items (in the top right corner).

### view

*string*

A frontend view.

### class

*string*

A CSS class or multiple classes for the LI tag.

### order

*integer*

An order position.

### disabled

*bool*

Whether is disabled.
