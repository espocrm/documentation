### How to build

Assuming you have *python* and *pip* installed.

Install the packages:

1. `pip install mkdocs`
2. `pip install mdx_truly_sane_lists`
3. `pip install mkdocs-material`

Command to build:

```
mkdocs build
```

On Windows:
```
python -m mkdocs build
```
